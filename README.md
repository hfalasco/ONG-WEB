## 📌 Sobre o Projeto
Este projeto foi desenvolvido como parte da disciplina de <b>Frontend</b>. O objetivo principal é criar uma ponte entre ONGs de animais da região de Ribeirão Preto e potenciais adotantes, focando na experiência do usuário e na simplicidade do processo de adoção.

## 🚀 Tecnologias Utilizadas
Para este projeto, mantivemos o "core" da web para dominar os fundamentos:

* **HTML:** Estruturação semântica de dados.
* **CSS:** Estilização, layout (Flexbox/Grid) e responsividade.
* **JavaScript** Lógica de interatividade e manipulação do DOM.
