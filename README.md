## 📌 Sobre o Projeto

Este projeto foi desenvolvido como atividade prática da disciplina de **Frontend**, integrante do curso de **Engenharia de Software**. O sistema tem como objetivo estabelecer uma ponte digital entre a ONG **Apoio à Proteção Animal de Guariba** e potenciais adotantes da região, promovendo o bem-estar animal por meio de uma interface web acessível, responsiva e intuitiva.

A proposta central é facilitar o processo de adoção responsável, oferecendo ao usuário uma experiência clara e humanizada no contato com os animais disponíveis para adoção.

---

## 🎯 Objetivos

- Aumentar a visibilidade da ONG na cidade de Guariba e região;
- Facilitar o acesso de potenciais adotantes às informações dos animais disponíveis;
- Simplificar o processo de contato entre a ONG e a comunidade;
- Aplicar na prática os conceitos de desenvolvimento web aprendidos em sala de aula.

---

## 🚀 Tecnologias Utilizadas

O projeto foi desenvolvido com as tecnologias fundamentais da web, priorizando o domínio dos fundamentos:

| Tecnologia | Finalidade |
|---|---|
| **HTML5** | Estruturação semântica do conteúdo |
| **CSS3** | Estilização, layout (Flexbox/Grid) e responsividade |
| **JavaScript** | Lógica de interatividade e manipulação do DOM |
